# 03_utils — Shared Utilities

**What it does:** Provides `logger.py` (logging setup), `time.py` (UTC helpers), `formatter.py` (metric/duration formatting), and `errors.py` (exception logging, safe_call, format_error). No business logic — pure helpers.

**Dependencies:** None (stdlib only).

**How to test:**
```
cd backend/foundation/03_utils
python test_utils.py
```

**Pass condition:**
```
utils module — ALL TESTS PASSED. Safe to proceed.
```
